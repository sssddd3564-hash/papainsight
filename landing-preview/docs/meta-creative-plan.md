# 파파컴퍼니 메타 광고 소재 기획안

## 1. 소재 목표

음식점 사장님이 인스타그램 피드에서 스크롤을 멈추고, 네이버 플레이스 상위노출 전략 진단을 신청하게 만드는 원페이지 광고 소재를 제작합니다.

소재는 카드뉴스가 아니라 한 장에서 아래 흐름이 끝나야 합니다.

```text
강한 후킹 → 문제 인식 → 무료 진단 제안 → 신청 행동
```

핵심 메시지는 `네이버 플레이스 상위노출 전략 제안`이며, 표현은 공격적으로 가되 `순위 보장`, `무조건 1위`, `매출 보장`처럼 리스크가 큰 문구는 사용하지 않습니다.

## 2. 제작 포맷

### 1차 제작

- 1:1 피드: 1080 x 1080
- 4:5 피드: 1080 x 1350

### 2차 확장

- 9:16 스토리/릴스: 1080 x 1920
- 1차 성과가 좋은 후킹 문구만 세로형으로 확장합니다.

## 3. 원페이지 레이아웃

한 장 안에서 메시지가 바로 읽혀야 합니다.

| 영역 | 비중 | 역할 |
| --- | --- | --- |
| 상단 | 20% | 강한 후킹 헤드라인 |
| 중앙 | 45% | 문제 상황 또는 사례형 비주얼 |
| 하단 | 25% | 진단 항목 3개와 CTA |
| 최하단 | 10% | 파파컴퍼니 로고, 무료 진단 문구 |

### 레이아웃 규칙

- 첫 문장은 1초 안에 읽혀야 합니다.
- 핵심 문장은 한 소재 안에 최대 2개만 사용합니다.
- CTA는 버튼처럼 보이게 만듭니다.
- 본문 설명은 길게 쓰지 않고 짧은 라벨로 처리합니다.
- 시선은 `헤드라인 → 비주얼 → CTA` 순서로 흐르게 합니다.

## 4. 디자인 톤

- 랜딩페이지와 동일한 보라 포인트 컬러를 사용합니다.
- 배경은 화이트 또는 연한 보라 톤을 사용합니다.
- 헤드라인은 검정 또는 아주 진한 남보라로 강하게 처리합니다.
- CTA 버튼은 보라색으로 처리합니다.
- 체크 아이콘, 검색 결과 카드, 지도 UI, 순위 그래프 같은 요소를 사용합니다.
- 음식 사진보다 `검색/지도/플레이스/순위` 맥락이 보이는 UI형 비주얼을 우선합니다.

## 5. 후킹 문구

### 손실 자극형

- `근처 손님을 경쟁 매장에 뺏기고 있을 수 있습니다`
- `광고비 쓰기 전에 플레이스부터 망가져 있을 수 있습니다`
- `손님은 검색하는데 우리 가게는 뒤에서 멈춰 있을 수 있습니다`
- `검색 노출이 약하면 좋은 메뉴도 발견되지 않습니다`

### 경쟁 비교형

- `같은 골목인데 왜 옆 가게만 먼저 보일까요?`
- `리뷰가 있어도 경쟁 매장에 밀릴 수 있습니다`
- `맛집 검색 결과에서 뒤로 밀리는 매장에는 이유가 있습니다`
- `상위 매장과 하위 매장은 보여주는 정보부터 다릅니다`

### 광고비 방어형

- `플레이스가 정리되지 않으면 광고비가 새어 나갑니다`
- `검색 노출이 약한 상태에서 광고만 키우면 손해입니다`
- `광고보다 먼저 봐야 할 건 플레이스 상태입니다`
- `광고 클릭 전에 고객은 이미 플레이스를 보고 판단합니다`

### 사례 기대형

- `플레이스 순위가 바뀌면 문의 흐름도 달라집니다`
- `노출이 막힌 지점을 찾으면 개선 방향이 보입니다`
- `상위노출 전략은 매장 상태를 보는 것부터 시작됩니다`
- `우리 매장에 맞는 플레이스 개선 방향을 받아보세요`

## 6. 소재별 제작 브리프

### 소재 A. 경쟁 매장 비교형

- 헤드라인: `같은 골목인데 왜 옆 가게만 먼저 보일까요?`
- 보조문구: `플레이스 노출 구조부터 무료로 진단합니다`
- CTA: `무료 진단 신청`
- 비주얼:
  - 검색 결과 카드 2개를 비교합니다.
  - 경쟁 매장은 상단에 선명하게 배치합니다.
  - 우리 매장은 아래쪽에 작게 배치합니다.
  - 보라색 화살표 또는 체크 표시로 개선 가능성을 암시합니다.
- 목적:
  - 음식점 사장님의 경쟁 불안을 자극합니다.
  - “우리 가게만 뒤에 있는 것 같다”는 문제 인식을 만듭니다.

### 소재 B. 손실 자극형

- 헤드라인: `근처 손님을 경쟁 매장에 뺏기고 있을 수 있습니다`
- 보조문구: `네이버 플레이스 상위노출 전략을 확인하세요`
- CTA: `상위노출 전략 받기`
- 비주얼:
  - 지도 또는 검색 화면 느낌을 사용합니다.
  - 고객 흐름이 경쟁 매장 쪽으로 가는 구도를 만듭니다.
  - 우리 매장은 화면 아래쪽 또는 흐린 영역에 배치합니다.
- 목적:
  - 놓치고 있는 근처 수요를 인식시킵니다.
  - 무료 진단을 손실 확인 행동으로 느끼게 합니다.

### 소재 C. 광고비 방어형

- 헤드라인: `광고비 쓰기 전에 플레이스부터 확인하세요`
- 보조문구: `노출을 막는 항목을 먼저 찾아드립니다`
- CTA: `무료 진단 받기`
- 비주얼:
  - 광고비 아이콘과 플레이스 체크리스트를 함께 배치합니다.
  - 돈이 새는 느낌은 과하지 않게 표현합니다.
  - 빨간 경고보다 보라색 체크/진단 톤을 사용합니다.
- 목적:
  - 광고비를 이미 쓰거나 고민하는 사장님을 겨냥합니다.
  - “광고 전에 기본 세팅이 먼저”라는 인식을 줍니다.

### 소재 D. 리뷰 역설형

- 헤드라인: `리뷰가 있어도 경쟁 매장에 밀릴 수 있습니다`
- 보조문구: `사진, 메뉴, 키워드, 운영 상태까지 함께 봅니다`
- CTA: `내 매장 진단하기`
- 비주얼:
  - 별점/리뷰 수는 높은데 검색 위치는 낮은 대비를 만듭니다.
  - 상단 매장과 하단 매장의 정보량 차이를 보여줍니다.
  - 체크 항목은 `사진`, `메뉴`, `키워드` 3개로 줄입니다.
- 목적:
  - 리뷰만 많으면 된다는 인식을 깨줍니다.
  - 플레이스 운영 요소를 함께 봐야 한다는 필요성을 만듭니다.

### 소재 E. 상위노출 전략형

- 헤드라인: `음식점 플레이스 상위노출, 전략 없이 어렵습니다`
- 보조문구: `우리 매장에 맞는 노출 개선 방향을 제안합니다`
- CTA: `전략 제안 받기`
- 비주얼:
  - 순위 상승 그래프를 사용합니다.
  - 플레이스 체크 항목 3개를 배치합니다.
  - 보라색 상승 화살표로 성과 기대감을 만듭니다.
- 목적:
  - 무료 진단의 결과물이 단순 점검이 아니라 전략 제안이라는 점을 강조합니다.
  - 랜딩페이지의 상위노출 메시지와 가장 직접적으로 연결합니다.

## 7. 제작 우선순위

1. 경쟁 매장 비교형
2. 손실 자극형
3. 광고비 방어형
4. 리뷰 역설형
5. 상위노출 전략형

초기에는 1~3번을 먼저 제작합니다. 성과가 가장 좋은 소재의 문구와 구도를 유지한 채, 4:5와 9:16 버전으로 확장합니다.

## 8. 소재 검수 기준

광고 업로드 전 아래 기준을 확인합니다.

- 모바일에서 헤드라인이 1초 안에 읽히는가?
- 한 장 안에서 문제와 제안이 모두 보이는가?
- CTA가 버튼처럼 명확한가?
- 파파컴퍼니 보라 톤이 유지되는가?
- 네이버 공식 제휴처럼 오해될 표현이 없는가?
- 순위/매출 보장처럼 심사 리스크가 있는 표현이 없는가?
- 랜딩페이지의 무료 진단 흐름과 연결되는가?

## 9. 금지 표현

아래 표현은 사용하지 않습니다.

- `순위 보장`
- `무조건 1위`
- `매출 보장`
- `네이버 공식`
- `100% 상위노출`
- `바로 1위`
- `확실한 매출 상승`

광고에서는 보장보다 `진단`, `전략 제안`, `개선 방향`, `노출 구조 확인` 중심으로 표현합니다.
